# 通讯录管理系统简介 java949


## 技术栈

本项目采用以下技术构建：

- 后端：Spring Boot
- 前端：Vue
- 数据库：MySQL

## 功能模块

### 用户角色

系统主要分为以下两种角色：

- 管理员
- 其他用户

### 功能列表

1. 注册与登录
   - 用户注册
   - 用户登录
2. 通讯录公告
   - 查看公告
3. 通讯录用户管理
   - 新增用户
   - 删除用户
   - 修改用户信息
   - 查询用户信息
4. 通讯录组管理
   - 编辑用户组（如小学组、初中组、高中组、大学组、家人组）
5. 通讯录菜单管理
   - 编辑通讯录组对应的菜单功能权限
6. 通讯录公告管理
   - 发布公告
   - 编辑公告
   - 删除公告
7. 个人信息管理
   - 修改个人用户信息（如用户头像）

## 系统运行环境

- 服务器：推荐使用 Linux 或 Windows 服务器
- 数据库：MySQL（版本 5.7+）
- 后端运行环境：JDK 1.8+
- 前端运行环境：现代浏览器（Chrome、Firefox、Safari 等）

## 部署步骤

1. 准备运行环境（服务器、数据库等）
2. 克隆项目代码
3. 部署后端（Spring Boot 应用）
4. 部署前端（Vue 应用）
5. 启动应用

## 目录结构

```
通讯录管理系统
|-- 后端（Spring Boot）
|-- 前端（Vue）
```

## 核心亮点

- 前后端分离，易于维护与扩展
- 基于角色的权限控制，安全性高
- 采用主流技术栈，便于二次开发与优化
